# ก.พ. 2569 Research Project

โปรเจกต์นี้มีวัตถุประสงค์เพื่อทำการค้นคว้า วิจัย และวิเคราะห์ข้อสอบ ก.พ. ภาค ก (Paper & Pencil) ประจำปี 2569 และรวบรวมข้อมูลสถิติ/แนวข้อสอบย้อนหลัง (2566 - 2568) รวมถึงรวบรวมเทคนิค วิธีคิด และแหล่งเรียนรู้จากผู้สอบจริง

## วัตถุประสงค์หลัก
1. ศึกษารายละเอียดโครงสร้างข้อสอบ ก.พ. ภาค ก ปีล่าสุด
2. วิเคราะห์แนวโน้มและวิชาที่ออกสอบย้อนหลัง (2566 - 2568)
3. รวบรวมข้อมูลจำโจทย์และประสบการณ์จากผู้สอบจริงบนโซเชียลมีเดีย (Facebook, Pantip, YouTube)
4. จัดทำแผนการเตรียมตัวสอบ แผนการอ่านหนังสือ 30 วัน และ Checklist ก่อนเข้าห้องสอบ
